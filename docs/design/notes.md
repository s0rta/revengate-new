## Character Archetypes
* The Innocent
-- Goal: To be happy
-- Flaws: Too trusting
-- Skill: Open minded

* The Orphan
-- Goal: To belong
-- Flaws: Cynical
-- Skill: Open, realistic

* The Hero
-- Goal: To belong
-- Flaws: Need to fight
-- Skill: Courage

* The Care Giver
-- Goal: To help others
-- Flaws: Martyrdom
-- Skill: Compassion

* The Explorer
-- Goal: A full life
-- Flaws: Wandering
-- Skill: Autonomy

* The Rebel
-- Goal: Change norms
-- Flaws: Crime
-- Skill: Wild freedom

* The Lover
-- Goal: Relationship
-- Flaws: Pleasing all
-- Skill: Friendship

* The Creator
-- Goal: Realizing a vision
-- Flaws: Perfectionism
-- Skill: Imagination

* The Jester
-- Goal: Lighten up
-- Flaws: Wasting time
-- Skill: Humour

* The Sage
-- Goal: Wisdom
-- Flaws: No action
-- Skill: Intelligence

* The Magician
-- Goal: Transforming
-- Flaws: Egostistical
-- Skill: Win/wins

* The Ruler
-- Goal: Prosperity
-- Flaws: Authoritarian
-- Skill: Leadeaship
